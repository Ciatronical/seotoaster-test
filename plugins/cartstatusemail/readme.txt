Cart Status Email plugin.

First click "Merchandising" on main panel. Then click 'Email remarketing';

And then fill next fields:

1. lag time - after this time email will be sent to the user
2. Cart status - choose cart status
3. Then choose if cart 'that includes' or 'regardless of cart content'
   if you choose 'that includes' :
    a) choose all/any
    b) click add products and choose products for cart
4. Choose email template. (By default plugin used email template from action emails 'Email remarketing' section)
5. Email from (By default plugin used 'Email from' from action emails 'Email remarketing' section)
6. Insert you message (By default plugin used 'message' from action emails 'Email remarketing' section)

                                    Action Email lexems
====================================================================================================

1. {unsubscribe:link} - unsubscribe link.
